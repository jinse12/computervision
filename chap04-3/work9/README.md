# 과제 9
키보드를 이용하여 직선을 그리는 프로그램을 만들고자 한다.
  - 초기 화면은 폭 500, 높이 500픽셀의 3채널 컬러영상으로 설정하라. 직선의 시작 위치는 화면의 중심으로 설정하고 'q'키를 누르면 종료하도록 하라.
  - 'l' 키를 누르면 현재위치에서 왼쪽으로 길이 50픽셀인 직선을 그려준다.
  - 'u' 키를 누르면 현재위치에서 위쪽으로 길이 50픽셀인 직선을 그려준다.
  - 'r' 키를 누르면 현재위치에서 오른쪽으로 길이 50픽셀인 직선을 그려준다.
  - 'd' 키를 누르면 현재위치에서 아래쪽으로 길이 50픽셀인 직선을 그려준다.
---
## 코드
```cpp
#include <opencv2/opencv.hpp>   // OpenCV 함수들을 사용하기 위한 헤더 파일
using namespace cv;             // cv::Mat, cv::line 등을 쓸 때 cv::를 생략하기 위해 사용

int main()
{
    Mat img(500, 500, CV_8UC3, Scalar(255, 255, 255));  // 500x500 흰색 영상 생성

    Point cur(250, 250);    // 시작 위치: 화면 중심

    while (true) {
        imshow("src", img);

        int key = waitKey(0);
        Point next = cur;   // 현재 위치에서 시작

        if      (key == 'l' || key == 'L') next.x -= 50;   // 왼쪽으로 50px
        else if (key == 'r' || key == 'R') next.x += 50;   // 오른쪽으로 50px
        else if (key == 'u' || key == 'U') next.y -= 50;   // 위쪽으로 50px
        else if (key == 'd' || key == 'D') next.y += 50;   // 아래쪽으로 50px
        else if (key == 'q' || key == 'Q') break;

        line(img, cur, next, Scalar(0, 0, 0), 1, LINE_AA); // 현재 위치 → 다음 위치 선 그리기
        cur = next;     // 끝점이 새 시작점
    }
    return 0;
}
```
---
## 개념
### 선 누적 그리기
`img` 를 루프 밖에서 한 번만 생성하므로 매 루프마다 초기화되지 않는다.
키를 누를 때마다 선이 누적되어 그려진다.
이전 문제(버튼 이동)와의 차이점이다.

| 방식 | img 생성 위치 | 결과 |
|------|--------------|------|
| 버튼 이동 | 루프 안 | 매번 초기화, 이전 내용 지워짐 |
| 선 그리기 | 루프 밖 | 누적, 이전 선 유지 |

### cur → next 패턴
`next = cur` 로 현재 위치를 복사한 뒤 키 입력에 따라 좌표를 수정한다.
`line(img, cur, next, ...)` 으로 선을 그린 후 `cur = next` 로 끝점을 새 시작점으로 갱신한다.

### Point — 좌표 연산
`Point` 는 x, y 좌표를 가지는 구조체로 `.x`, `.y` 로 직접 접근해 값을 수정할 수 있다.

| 키 | 동작 |
|----|------|
| `l` / `L` | `next.x -= 50` (왼쪽) |
| `r` / `R` | `next.x += 50` (오른쪽) |
| `u` / `U` | `next.y -= 50` (위쪽) |
| `d` / `D` | `next.y += 50` (아래쪽) |
| `q` / `Q` | 종료 |
