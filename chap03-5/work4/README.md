# 과제 4

배경색이 흰색인 400X400 영상을 만들고 y = 1/400 * x² 의 그래프를 그리시오

---

## 코드

```cpp
#include <opencv2/opencv.hpp>   // OpenCV 함수들을 사용하기 위한 헤더 파일
#include <iostream>             // 입출력(printf, cout 등)을 사용하기 위한 헤더 파일
using namespace cv;             // cv::Mat, cv::imshow 등을 쓸 때 cv::를 생략하기 위해 사용
using namespace std;            // std::cout, std::endl 등을 쓸 때 std::를 생략하기 위해 사용

int main() {
    // 400x400 크기의 1채널 이미지 생성, 모든 픽셀을 255(흰색)으로 초기화
    Mat img(400, 400, CV_8UC1, Scalar(255));

    // x를 0부터 이미지 가로 끝까지 반복
    for (int x = 0; x < img.cols; x++) {

        // y = 1/400 * x² 공식 적용
        // 1.0/400.0 : 정수 나눗셈 방지를 위해 실수로 계산
        // (int) : 계산 결과를 정수(픽셀 좌표)로 변환
        int y = (int)(1.0 / 400.0 * x * x);

        // (y, x) 위치의 픽셀을 0(검은색)으로 설정 -> 포물선 점 찍기
        // at<uchar> : 그레이스케일이므로 uchar 타입 사용
        // 첫 번째 인자 y = 행(row), 두 번째 인자 x = 열(col)
        img.at<uchar>(y, x) = 0;
    }

    // "img" 창에 결과 이미지 출력
    imshow("img", img);

    // 키 입력이 있을 때까지 대기
    waitKey();

    return 0;
}
```

---

## 개념

### 1. 이미지 생성
| 인자 | 의미 |
|------|------|
| `400, 400` | 이미지 크기 (행 400, 열 400) |
| `CV_8UC1` | 8비트 1채널 (그레이스케일) |
| `Scalar(255)` | 모든 픽셀을 255(흰색)으로 초기화 |

### 2. 포물선 공식
| 표현 | 설명 |
|------|------|
| `1/400 * x²` | 정수 나눗셈 → 결과가 항상 0 |
| `1.0/400.0 * x * x` | 실수 나눗셈 → 정확한 계산 |

### 3. 픽셀 접근
| 코드 | 의미 |
|------|------|
| `at<uchar>(y, x)` | (y행, x열) 픽셀에 접근 |
| `= 0` | 해당 픽셀을 검은색으로 설정 |
| `= 255` | 해당 픽셀을 흰색으로 설정 |

### 4. 세로선 vs 가로선 원리
| 선 종류 | 고정 | 반복 |
|---------|------|------|
| 세로선 | col(열) 고정 | row(행) 반복 |
| 가로선 | row(행) 고정 | col(열) 반복 |
