## **1. 컴퓨터비전 기술이 적용된 제품을 찾고 제품의 기능, 동작원리 등을 조사하여 제출하시오.**
  - 자동차 번호판 인식 시스템
     - 공장 내 차량 출입 관리, 물류 추적, 보안 강화를 위해 차량 번호판을 실시간으로 인식하고 데이터베이스와 연동하여 차량 정보를 자동으로 확인하는 시스템이다.
     - 동작 원리는 고해상도 카메라로 번호판 이미지를 캡처한 뒤, 컴퓨터 비전 알고리즘을 통해 문자 영역을 추출하고 광학 문자 인식(Optical Character Recognition) 기술로 문자를 판독한다. 또한 딥러닝 모델을 활용하여 다양한 조명, 각도, 날씨 조건에서도 인식 정확도를 높이는 방식으로 작동한다.

## **2. 영상처리, 컴퓨터비전의 차이점을 자세히 설명하시오.**  
- 영상처리는 입출력이 모두 영상인 형태이며 이미지를 입력으로 받아 화질개선, 잡음제거, 기하학적 변환 등 처리로 새로운 영상을 만드는 작업이다.  
- 컴퓨터비전은 사람의 시각기능을 컴퓨터로 대체하여 영상으로부터 원하는 정보를 인식하는 작업이다.

## **3. CMOS센서의 원리를 조사하시오.**
- CMOS 센서는 빛을 전기 신호로 바꾸는 이미지 센서로, 각 픽셀에 광다이오드와 트랜지스터가 포함되어 있다.  
렌즈로 들어온 빛이 광다이오드에 닿으면 광전 효과로 전하를 생성하고, 빛 세기에 비례한 전하량이 축적된다.  
이 전하를 픽셀 내 트랜지스터가 전압 신호로 바꾼 뒤, AD 변환기로 디지털 데이터로 처리해 이미지를 만든다.

## **4. 영상의 해상도와 이미지 센서의 개수의 관계를 설명하시오.**
- 영상의 해상도는 이미지 센서의 픽셀 개수에 직접적으로 비례한다.  
센서에 픽셀이 많을수록 더 세밀한 이미지를 구성할 수 있어 해상도가 높아진다.

## **5. 100X100 그레이스케일 영상 파일의 크기(바이트)를 계산하라.**
- 100×100 그레이스케일 영상은 총 10,000 픽셀을 가지며, 각 픽셀이 256 단계로 밝기를 표현한다.  
따라서 파일의 크기는 10,000 픽셀 x 1바이트/픽셀 = 10,000 바이트이다.

## **7 100X100 컬러 영상 파일의 크기(바이트)를 계산하라.**
- 컬러영상은 하나의 픽셀당 R, G, B 3개의 값이 저장된다. 컬러 영상 파일 크기는  
픽셀 수 x 채널 수 x 바이트 = 10,000 x 3 x 1 = 30,000 바이트이다
