## **1. OpenCV를 이용하여 개발된 제품을 조사하고 기능을 설명하라.**
   - 차량 번호판 인식 시스템   
        - 주로 주차장 출입 시스템, 교통 단속, 보안 시스템 등에 사용됩니다.
        1. 카메라를 통해 차량 이미지를 획득하고, 차량 이미지에서 경계선 검출, 윤곽선 탐지, 딥러닝 기반 객체 탐지 과정으로 번호판 영역을 검출합니다.
        2. 인식률을 높이기 위해 흑백 변환, 노이즈 제거, 이진화 과정을 거치는 이미지 전처리 후 번호판에서 각 문자와 숫자를 분리합니다.
        3. 광학 문자 인식 기술(OCR)로 번호판의 문자와 숫자를 인식합니다.

## **2. 소프트웨어 분야의 용어인 라이브러리의 의미를 자세히 설명하라.**
  - 라이브러리는 자주 사용되는 기능들을 함수/클래스/모듈로 만들어 놓은 것으로, 헤더파일과 목적파일로 구성되어 있습니다.   
   개발자가 반복적으로 구현하지 않고 호출하여 사용합니다.

## **3. 오픈소스의 의미를 설명하라.**
  - 오픈소스는 소프트웨어의 소스코드를 공개하여 누구나 자유롭게 검토, 수정, 배포할 수 있는 소프트웨어를 의미합니다.

## **4. Windows 운영체제의 환경변수 중 Path 변수의 기능을 설명하라.**
  - Windows 운영체제의 환경 변수 중 PATH는 명령 프롬프트나 셸에서 실행 파일(프로그램)을 검색할 때 참조하는 디렉토리 경로 목록입니다.   
    사용자가 명령어를 입력하면, Windows는 현재 디렉토리와 PATH에 지정된 디렉토리를 순서대로 검색해 해당 실행 파일을 찾습니다. 이를 통해 전체 경로를 입력하지 않고 간단히 명령어를 실행할 수 있습니다.
    
## **5. 프로그램 개발과정인 편집, 컴파일, 링크, 실행 단계를 각각 설명하라. 특히, 링크단계에서 라이브러리의 역할에 대하여 설명하라.**
  - 편집 단계 : 텍스트 에디터나 IDE를 사용해 소스 코드를 작성하는 단계입니다. 변수 선언, 함수 정의가 포함됩니다.
  - 컴파일 단계 : 소스 코드를 기계어(오브젝트 코드)로 변환하는 과정입니다. 컴파일러(예: gcc)가 소스 파일(.c, .cpp)을 검사해 문법 오류를 검출하고, 컴파일이 완료되면 목적 파일(.o or .obj)을 생성합니다.
  - 링크 단계 : 컴파일된 목적 파일과 필요한 라이브러리를 연결하여 하나의 실행 파일을 만드는 과정입니다. 라이브러리는 미리 컴파일된 함수나 코드를 제공합니다.   
프로그램에서 라이브러리 함수를 호출하면, 링커가 해당 라이브러리를 찾아 연결합니다.   
코드 재사용이 가능하며, 개발 시간을 단축할 수 있고, 표준 라이브러리나 외부 라이브러리를 포함합니다.

## **6. 정적 라이브러리(SLL)와 동적 라이브러리(DLL)에 대하여 조사하라. 특히, 각 라이브러리가 실행파일에 링크되는 시점을 자세히 설명하라.**
  - 정적 라이브러리는 링크 단계에서 목적파일과 결합하여 실행파일을 생성하며, 동적 라이브러리는 실행 단계에서 실행 파일과 연결되어 실행됩니다.

## **7. 동적 라이브러리(DLL)의 경로는 운영체제의 환경변수 PATH에 등록되어야 한다. 이유를 설명하라.**
  - 프로그램이 실행되지 않거나 오류가 발생할 수 있기 때문입니다.

   동적 라이브러리(DLL)는 프로그램 실행 시 필요한 함수나 기능을 제공하는 외부 파일입니다. 프로그램이 실행될 때 운영체제는 필요한 DLL 파일을 찾아 프로그램에 연결하여 사용합니다.
하지만 DLL 파일이 프로그램과 같은 폴더에 있지 않은 경우, 운영체제는 정해진 경로를 검색하여 DLL을 찾습니다. 이때 사용되는 것이 운영체제의 환경변수 PATH입니다.

   환경변수 PATH에는 실행 파일이나 라이브러리를 검색할 수 있는 디렉터리 경로 목록이 저장되어 있습니다. 프로그램이 실행되면 운영체제는 PATH에 등록된 폴더들을 순서대로 검색하여 필요한 DLL 파일을 찾습니다.

   따라서 DLL 파일이 있는 폴더를 PATH에 등록하지 않으면 운영체제가 해당 라이브러리를 찾지 못해 프로그램 실행 시 오류가 발생할 수 있습니다. 반대로 DLL 경로를 PATH에 등록해 두면 운영체제가 해당 위치에서 라이브러리를 쉽게 찾을 수 있어 프로그램이 정상적으로 실행됩니다.
